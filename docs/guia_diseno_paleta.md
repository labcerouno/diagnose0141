# Guia de diseno y paleta

## Enfoque
- La interfaz es mobile-first.
- El layout principal debe funcionar sin panel lateral.
- Estetica: premium, profesional, orientada a CEOs.

## Tipografia
- Fuente principal: Miranda Sans.
- Usar pesos medios/semibold/bold para jerarquia ejecutiva.
- Mantener textos con tamano ligeramente mayor para lectura comoda en mobile.

## Paleta oficial (uso exclusivo)
- #222B2E
- #CBCBD0
- #7B818C
- #9599A2
- #FAFAFA
- #35424C
- #6CC5DA
- #E52E34

## Regla de color
- No usar colores fuera de esta paleta.
- Se permiten variaciones alpha (rgba) derivadas solo de estos colores.
- Mantener alto contraste para lectura y claridad de decision.
